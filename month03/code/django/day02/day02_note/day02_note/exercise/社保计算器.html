社会保险的主要项目包括养老保险、医疗保险、失业保险、工伤保险、生育保险。

五险:

  • 养老
  • 失业
  • 工伤
  • 医疗
  • 生育

一金:

  • 公积金

2016年度北京市职工月平均工资为7706元

社保基数为每个人缴纳社保的基准数：
  • 最小值为上年月平均工资的40%即7706*40%=3082元
  • 最大值为上年月平均工资的300%即7706*3=23118元
